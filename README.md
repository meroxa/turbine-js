# DAFT-func -- Data Application Framework To Functions 

## To run sample app in local mode ("I am just locally testing that my data app works")
Prereqs
+ Node
+ Yarn

```
git clone git@github.com:meroxa/daft-func.git
cd daft-func
yarn install
```

Then run:
```
node index.js
```
